Trabajo de Investigación: Uso de display:grid y Posicionamiento

1. ¿Qué es display: grid?

Grid es una forma de maquetar elementos en dos dimensiones (filas y columnas). Es ideal para estructuras más complejas donde se requiere control total sobre la posición de los elementos.

2. Propiedades utilizadas en Grid:

  • display: grid
  • grid-template-columns y grid-template-rows
  • grid-column y grid-row
  • gap

3. Comparación con Flexbox

Flexbox es ideal para layouts de una dimensión (una sola fila o columna). Algunas propiedades que comparten Flex y Grid:

  • gap (funciona en ambos)
  • justify-content, align-items
  • order para reordenar elementos

4. Posicionamiento adicional

También se puede usar position: relative, absolute o fixed para ajustar elementos dentro del Grid o fuera de él.

5. Conclusión

El uso de Grid facilita la organización de elementos complejos en dos dimensiones. Además, se pueden combinar comandos de Flexbox o posicionamiento (relative, absolute) para lograr diseños más flexibles y precisos.
